Crypto Risk-Neutral Density

Extracted market-implied probability distributions of future BTC and ETH prices from live Deribit options data using the Breeden-Litzenberger theorem.

PythonBlack-ScholesBreeden-LitzenbergerDeribit APIPlotlySciPy

What it does

Options markets encode collective beliefs about future asset prices. By observing how implied volatility varies across strikes (the volatility smile), this project recovers the full probability distribution the market is pricing in, without assuming any particular model.

The pipeline has three steps: pull the live option chain from Deribit's API across all expiries, apply Breeden-Litzenberger to extract the risk-neutral density (Q-measure), then estimate the real-world density (P-measure) from historical log-returns via CoinGecko and compare the two. Because the second derivative of noisy options data is unstable, a degree-4 smoothing spline is first fit to the implied volatility surface before differentiating.

Why it matters

The gap between Q and P is economically meaningful: it reflects the variance risk premium and crash risk aversion embedded in options prices. For BTC, the Q mean sits ~$2,500 above the P mean, showing options buyers systematically pay a premium for protection. Running this across all available expiries produces a full term structure of market beliefs, visualized as an interactive 3D density surface.

The Math

Breeden-Litzenberger (1978)

q(K) = erT · ∂²C/∂K²

The risk-neutral density is the second derivative of call prices w.r.t. strike, discounted at the risk-free rate.

Black-Scholes Call Price

C = S·Φ(d₁) − Ke−rT·Φ(d₂)

Used to convert the smoothed implied volatility surface to call prices before differentiating.

Real-World (P) Density

log(ST/S₀) ~ N(μT, σ²T)

μ and σ estimated from 365 days of daily log-returns via CoinGecko, then annualized.

Converts raw option prices into a probability density function by taking the second derivative of the call price w.r.t strike (Breeden-Litzenberger).

  def compute_rnd(df, S, T, r=0.05):
      """
      Breeden-Litzenberger: q(K) = e^(rT) * d²C/dK²
      1. Fit a smoothing spline to the Implied Volatility surface.
      2. Convert smoothed IV back to Call Prices using Black-Scholes.
      3. Take the second derivative of the Call Price curve.
      """
      
      # 1. Fit smoothing spline to implied vol surface
      vol_spline = UnivariateSpline(
          otm['strike'].values, otm['sigma'].values,
          k=4, s=len(otm) * 0.001
      )
      
      # 2. Convert smoothed IV → call prices via Black-Scholes
      K_grid     = np.linspace(strike_min, strike_max, 500)
      sigma_grid = np.clip(vol_spline(K_grid), 0.01, None)
      call_grid  = np.array([black_scholes_call(S, K, T, r, sig)
                             for K, sig in zip(K_grid, sigma_grid)])
  
      # 3. Differentiate twice → risk-neutral density
      call_spline = UnivariateSpline(K_grid, call_grid, k=4, s=0)
      d2C_dK2     = call_spline.derivative(n=2)(K_grid)
  
      # Discount to present value
      density = np.exp(r * T) * d2C_dK2
      density = np.clip(density, 0, None) 
      
      return pd.DataFrame({'strike': K_grid, 'density': density})
Pulls the full option chain from Deribit for a specific expiry and filters for active instruments to ensure liquidity.

  def get_option_chain(target_date_str):
      """Pull all BTC options for a given expiry from Deribit API."""
      instruments = get_btc_instruments()
      
      # Filter for options expiring on the target date
      target_date = pd.Timestamp(target_date_str).date()
      filtered    = [
          i for i in instruments
          if pd.to_datetime(i['expiration_timestamp'], unit='ms').date() == target_date
      ]
      
      rows = []
      for inst in filtered:
          # Fetch live ticker data (Mark Price, IV)
          ticker = requests.get(
              "https://www.deribit.com/api/v2/public/ticker",
              params={"instrument_name": inst['instrument_name']}
          ).json()['result']
          
          rows.append({
              'strike':      inst['strike'],
              'option_type': inst['option_type'],
              'implied_vol': ticker.get('mark_iv'),
              'mark_price':  ticker.get('mark_price'),
          })
          
      return pd.DataFrame(rows).dropna(subset=['implied_vol'])

How it was built
1
Step 01

Design first

Picked the colors, fonts, and spacing before writing any HTML. Easier to code toward something than to figure it out as you go.

2
Step 02

One stylesheet

All shared styles in styles.css. Each project page only adds its own styles in a small inline block at the top.

3
Step 03

Project pages

Each project is its own HTML file with a hero, sticky tabs, and content panels. Tab switching is about 10 lines of JavaScript.

4
Step 04

Live embeds

The Spotify and Crypto projects have Streamlit apps embedded via iframes on a dedicated Live App tab.

5
Step 05

GitHub Pages

Static files, no build step. Push to main and it's live within a minute.
